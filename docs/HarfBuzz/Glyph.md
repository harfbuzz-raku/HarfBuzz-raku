### has Associative %!ast

Glyph summary hash

Of the form `%( :$ax, :$ay, :$dx, :$dy, :g($gid), :$c)`

where

  * *dx* - pre X offset

  * *dy* - pre Y offset

  * *ax* - post X advance

  * *ay* - post Y advance

  * *g* - Glyph ID

  * *c* - Input character position
